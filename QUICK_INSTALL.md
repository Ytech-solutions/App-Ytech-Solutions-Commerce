# 🚀 Installation One-Click

## 🪟 Windows
Double-cliquez sur `install.bat` → **30 secondes** → ✅ **Terminé !**

## 🍎 macOS/Linux  
```bash
chmod +x install.sh && ./install.sh
```

---

## 🎯 Ce qui est installé automatiquement :
- ✅ Base de données `ytech_commerce`
- ✅ 6 tables complètes
- ✅ 4 packs de démonstration
- ✅ Fonctions triggers
- ✅ Index optimisés

## 🔧 Après installation :
1. Configurez `.env` avec vos credentials PostgreSQL
2. `npm install`
3. `npm run dev`

**🌐 Votre site sera prêt sur http://localhost:3000**
